# BizFlow - Sistema de Gestão Integrado

Sistema completo de gestão empresarial com frontend e backend integrados.

## 🚀 Funcionalidades

- **Autenticação Segura** - Login com JWT e 2FA
- **Gestão de Produtos** - Cadastro, estoque e categorias
- **PDV (Ponto de Venda)** - Vendas rápidas e eficientes
- **Gestão de Clientes** - CRM integrado
- **Controle Financeiro** - Receitas, despesas e relatórios
- **Gestão de Documentos** - Faturas, contratos e assinaturas digitais
- **Inventário** - Controle de stock em tempo real

## 🛠️ Tecnologias

### Frontend
- HTML5, CSS3, JavaScript Vanilla
- Design responsivo
- Interface moderna e intuitiva

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT para autenticação
- Bcrypt para hash de senhas
- Validação de dados robusta
